# KanJect

## Description
KanJect is a tool designed for managing and interacting with KAN (Kolmogorov-Arnold Network) models. It focuses on simplifying the injection, fine-tuning, or architectural adjustment of these networks, enabling users to efficiently incorporate KAN-based layers into their existing machine learning pipelines.

## Features
* **Easy Integration:** Seamlessly add KAN layers to standard PyTorch or deep learning workflows.
* **Modular Architecture:** Components are designed for high customizability, allowing users to tweak activation functions and network topology.
* **Performance Focused:** Optimized for efficiency when dealing with Kolmogorov-Arnold Network operations.
